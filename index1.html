2- ¿Qué es el HTML? ¿En qué consiste?

Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.
Sus contenidos podrían ser párrafos, una lista con viñetas, o imágenes y tablas de datos.

3- ¿Qué es una etiqueta? ¿Funciona si se escribe tanto en mayúscula como en minúscula? ¿Cuál es la mejor práctica para escribir etiquetas?

Son códigos utilizados para "marcar" el texto de una página web, y dar instrucciones al navegador sobre cómo mostrarlo.
Si funciona si se escribe en mayuscula o en minuscula y la mejor practica seria escribir todas las etiquetas en minusculas para mantener una mejor legibilidad a la hora de escribir.

4- ¿A qué llamamos elemento HTML? ¿Cuál es su anatomía? Explicar sus partes.

El elemento "" encierra todo el contenido de la página entera y se le conoce como el elemento raíz (root element).
El elemento HTML establece dónde comienza o empieza a tener efecto el elemento y sus partes:

La etiqueta de apertura: consiste en el nombre del elemento (en este caso, p), encerrado entre paréntesis angulares de apertura y cierre.

El contenido: Este es el contenido del elemento. En este ejemplo, es el texto del párrafo.

La etiqueta de cierre: Es lo mismo que la etiqueta de apertura, excepto que incluye una barra diagonal antes del nombre del elemento. Esto indica dónde termina el elemento.

11- ¿Para qué sirve la etiqueta img? ¿Cuáles son sus atributos? ¿Para qué sirven cada uno de ellos?

"img" representa una imagen en el documento.
Cuenta con dos atributos, “src” y “alt”. “src” es para mostrar en dónde se encuentra la imagen. Y "alt" está definido en un conjunto de etiquetas para proporcionar un texto equivalente al objeto.
